How it works

Welcome to BugYou! Here’s how to become a debugging wizard :

🔍 Objective

You’ll be given a prewritten code snippet that contains one or more bugs. Your job is to:

  • Analyze the code carefully
  • Fix any errors (logical, syntax, or runtime)
  • Ensure all test cases pass

💡 Hints

You can request hints during a challenge. However:

  • Each hint you take deducts points
  • Use them wisely — your final score depends on it!

🎯 Scoring System

  • Correct fix without hints: 10 points
  • Each hint used: -2 points
  • Wrong submission: -5 points
  • All test cases must pass to score

Tips for Success

  • Read the test case descriptions before editing
  • Use print/debug statements to trace logic
  • Think like a compiler — what would break?
